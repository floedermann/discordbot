# discordbot

1.) Register a bot at discord developer site
2.) Copy config.py.sample to config.py
3.) Update the TOKEN in config.py
4.) Create a virtualenv or just do a pip3 -i requirements.txt
5.) Start the bot with python3 main.py
6.) Invite bot with the URL https://discordapp.com/oauth2/authorize?client_id=YOURBOTID&scope=bot
